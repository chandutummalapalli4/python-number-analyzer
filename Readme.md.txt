# 🔢 Python Number Analyzer

A Python console application that analyzes a given number and displays various mathematical properties such as palindrome, prime, Armstrong, Harshad, spy number, digit statistics, and more.

---

## 📌 Features

- 🔄 Reverse a Number
- 🔢 Count Total Digits
- ➕ Calculate Sum of Digits
- ✖️ Calculate Product of Digits
- 🔺 Find Largest Digit
- 🔻 Find Smallest Digit
- 🔵 Count Even Digits
- 🟢 Count Odd Digits
- 0️⃣ Check if Number Contains Zero
- 🔁 Check Palindrome Number
- 🧮 Check Prime Number
- 💪 Check Armstrong Number
- 🕵️ Check Spy Number
- ➗ Check Harshad Number

---

## 🛠️ Technologies Used

- Python 3
- While Loop

- Mathematical Operations

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/python-number-analyzer.git
```

2. Navigate to the project folder

```bash
cd python-number-analyzer
```

3. Run the program

```bash
python main.py
```
